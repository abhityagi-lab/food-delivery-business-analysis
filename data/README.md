# Dataset Description

This folder contains the dataset used for the Food Delivery Time Prediction and Business Analysis project.

Dataset Name: updated.csv

## Description
The dataset contains information about food delivery operations including delivery personnel details, traffic conditions, weather conditions, vehicle type, and delivery time.

The dataset is used to analyze factors affecting food delivery performance and to build a machine learning model to predict delivery time.

## Key Features

- Delivery_person_Age: Age of the delivery driver
- Delivery_person_Ratings: Ratings given to the delivery driver
- Weather_conditions: Weather conditions during delivery
- Road_traffic_density: Traffic density during delivery
- Vehicle_condition: Condition of the delivery vehicle
- Type_of_vehicle: Vehicle used for delivery
- Type_of_order: Type of order (Snack, Meal, Drinks)
- Time_taken(min): Total time taken to complete the delivery

## Purpose

The dataset is used for:

- Data cleaning and preprocessing
- Exploratory data analysis
- Visualization
- Machine learning model development
- Business insight generation
